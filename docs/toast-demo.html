Material Design 3 Features

  • Bottom-center positioning (Material Design standard)
  • Smooth slide-up animation with cubic-bezier easing
  • Dark background with high contrast text
  • Semantic color coding (success, warning, error, info)
  • Optional action buttons with uppercase text
  • Progress bar for auto-dismiss timing
  • Accessible with ARIA attributes
  • Responsive design for mobile devices
  • Stacking support for multiple toasts
  • Smooth dismiss animation

Usage Examples

Basic Toast

createToast({ message: 'File saved successfully', type: 'success' });

Toast with Title and Action

createToast({ title: 'Connection Lost', message: 'Unable to reach the server', type: 'error', action: { text: 'Retry', onClick: () => reconnect() }, duration: 0 // Persistent });

Helper Functions

// Quick helpers showSuccessToast('Operation completed'); showErrorToast('Something went wrong'); showWarningToast('Please review your input'); showInfoToast('New update available');
